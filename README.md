# 🧀  E-Commerce Backend

A modern **e-commerce backend** built with **ASP.NET**, following **Clean Architecture**, **CQRS**, and **DDD principles**.

This project simulates a real-world online food marketplace (cheese, olives, honey, butter, breakfast products, etc.).

The goal of this project is to demonstrate **scalable backend architecture, domain-driven design, and modern .NET development practices.**

---

# 🚀 Tech Stack

* ASP.NET Web API (.NET 8)
* Clean Architecture
* CQRS (MediatR)
* Domain Driven Design (DDD)
* Entity Framework Core
* PostgreSQL
* FluentValidation
* AutoMapper
* Serilog
* JWT Authentication

# 🧠 Architecture

This project follows **Clean Architecture** principles.

Layers:

* Domain
* Application
* Infrastructure
* Persistence
* API

CQRS pattern is implemented using **MediatR**.
